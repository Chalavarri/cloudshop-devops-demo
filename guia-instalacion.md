# Guía de instalación

## Requisitos previos

- Tener Git instalado
- Tener una cuenta en GitHub
- Tener Visual Studio Code

## Pasos de instalación

1. Clonar el repositorio
2. Abrir el proyecto en VS Code
3. Verificar los archivos del proyecto

## Validación

- Confirmar que el README.md exista
- Verificar que Git detecte los cambios
- Comprobar que el proyecto funcione correctamente
